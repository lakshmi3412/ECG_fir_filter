ECG FIR Filter for Noise Removal

Overview

This project implements a Finite Impulse Response (FIR) filter for ECG signal denoising. The filter was designed using MATLAB and implemented using Verilog HDL. Synthesis and physical design were performed using Synopsys tools.

Objectives

- Remove noise from ECG signals.
- Design and implement an FIR filter.
- Verify functionality through simulation.
- Perform synthesis and physical design flow.

Tools Used

- MATLAB
- Verilog HDL
- Synopsys Design Compiler (DC)
- Synopsys IC Compiler II (ICC2)

Design Flow

1. FIR filter design in MATLAB.
2. Verilog HDL implementation.
3. Functional verification.
4. Synthesis using Synopsys Design Compiler.
5. Physical design using Synopsys ICC2.
6. Performance analysis and result verification.

Project Features

- ECG signal noise reduction.
- FIR filter hardware implementation.
- Synthesis and physical design flow.
- Area and timing analysis.

Results

The FIR filter successfully removes noise from the ECG signal and improves signal quality. The Verilog design was synthesized and implemented using Synopsys tools.
